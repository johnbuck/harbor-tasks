Use your **browser tool** to look up a fact on a live website.

Open **`https://quotes.toscrape.com/js/`** in the browser. This site lists famous
quotes across several pages. The quotes are rendered by JavaScript, so you must
actually load the page in the browser (a plain HTTP fetch of the URL will not
show them), and you may need to move through more than one page to find the one
you need.

Find the quote that contains this exact phrase:

> *"give a stupid or misinformed beholder a black eye"*

**Report the author that THIS WEBSITE attributes the quote to** — not whoever you
think actually said it; use the attribution shown on the page itself.

Write your answer to **`/app/answer.txt`** as just the author's name, nothing
else (for example: `Ada Lovelace`).
