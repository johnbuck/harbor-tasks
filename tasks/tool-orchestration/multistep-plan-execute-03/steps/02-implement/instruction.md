Create `/app/c2f.py` so that:

```
python /app/c2f.py <celsius>
```

converts the Celsius value to Fahrenheit and prints it to exactly one decimal place.

Examples:
- `python /app/c2f.py 100` → `212.0`
- `python /app/c2f.py 0` → `32.0`
- `python /app/c2f.py -40` → `-40.0`

Requirements:
- Use only Python standard library modules (`sys`).
- Output must be exactly one decimal place with no extra text.
