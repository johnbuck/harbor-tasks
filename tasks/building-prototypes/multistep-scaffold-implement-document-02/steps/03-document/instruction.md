# Step 3 — Write the README

Write a `/app/README.md` that documents the `textkit` package.

The README must include:
- A brief description of what the package does.
- Installation or setup notes (if any).
- API reference for the `slugify(s)` function, including its parameter, the transformation rules, and return value.
- A runnable usage example showing `slugify` being called and the result printed.
