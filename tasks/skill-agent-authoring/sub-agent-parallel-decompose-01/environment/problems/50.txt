First, a capacity check applies — if the running total is GREATER THAN 40, twelve returned books are removed; OTHERWISE five more returned books are added. Next, the running total is tripled by an overnight consolidation. After that, a delivery brings 7 more returned books. Later, the crew restocks five additional returned books. Soon after, a parity rule applies — if the running total is ODD at that point, seven more returned books are added; if it is EVEN, exactly half of them are removed. By midday, fourteen returned books arrive from a partner site. The city library is staffed by 48 people that morning. Finally, look up the calibration offset published for reference code CAL-50 in the site registry and add that offset to the running total. How many returned books remain at the very end? Respond with the single final integer only.
