# Tools — notes (guidance only; does not control tool availability)

- **Memory:** `recall` (temporal knowledge-graph memory) and `hindsight`
  (learning memory) are wired as MCP tools. Use them to retrieve prior context
  at task start and to store durable facts as you work.
- **Filesystem / terminal / search:** available for inspecting and modifying the
  task environment. Prefer reading and running over guessing.
- Tool availability is configured in `openclaw.json` / plugins — not here. This
  file is just orientation.
